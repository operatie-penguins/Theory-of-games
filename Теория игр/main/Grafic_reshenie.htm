Методы решения матричных игр


     Методов решения матричных игр может быть много. В данном разделе рассматриваются наиболее часто используемые методы решения матричной игры:

          Правило доминирования - позволяет сократить размеры матрицы игры. С помощью этого правила находят некоторые чистые стратегии, которые не могут внести никакого вклада в искомые оптимальные смешанные стратегии.
          Графические методы - позволяют решать матричные игры с определенным видом матрицы игры: игры 2 x n, игры m x 2. Это эффективный метод, основанный на простых геометрических соображениях.
          Итерационный метод - этот метод всегда, независимо от вида и размера матрицы игры, дает возможность находить ориентировочное значение цены игры и доли чистых стратегий (оптимальные смешанные стратегии). Итерационный метод прост и универсален, объем и сложность вычислений сравнительно слабо растут по мере увеличения числа стратегий игроков.
          Сведение матричной игры к задаче линейного программирования - решение любой матричной игры может быть найдено методами линейного программирования.
