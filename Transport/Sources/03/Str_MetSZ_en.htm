Northwest corner method

Conditions of the problem can be conveniently written in matrix form of transport. Cells of the matrix of transportations, in which xij>0 called basic, and the rests xij=0 are free. The matrix has m + n-1 basic cells. Their number is equal to the number of independent equations, constraints. The value xij in the matrix of transportations is given by xij=min (residue cargo at the point Ai, unmet needs at Bj ). The value xij=0 in a free cell is not written explicitly, but instead it put a dash.
